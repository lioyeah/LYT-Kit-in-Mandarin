# 成果 🏋️

> 成果让想法得以呼吸，项目却让它们窒息。

您可以使用此数据视图来编译任何带有 `#effort` 标签的笔记。

``` dataview
TABLE WITHOUT ID
 file.link as Efforts,
 rank as "Rank"

FROM #effort

SORT rank desc


```

## 为什么「成果」是解放性的

将你的努力看作「成果」，而不是项目，是值得和解放性的，因为：

-   **定义明确**
    -   成果需要一些*努力*。当你将一个笔记标记为成果时，你让自己知道某种努力是有意图的。
-   **定义不是太清晰**
    -   成果有*定义上的余地*。像「项目」这样的术语过于狭隘，无法定义成果的广度。这种模糊性是一个特点，而不是一个错误。
